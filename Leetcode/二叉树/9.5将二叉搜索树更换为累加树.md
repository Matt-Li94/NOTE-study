# 将二叉树更换为累加树
这道题目就是将节点上的数值变为前面的累加值而已
难度不大 观察题目给出的例子就是按照右中左的遍历就可以了
值得注意的是 你犯了一个错误 你需要记住的是 在你的遍历过程中 当前节点的值就已经是累加值了 不需要再加上sum了
```cpp
class Solution {
public:
    void traversal(TreeNode* & root, int &sum) {
        if(root == nullptr) return;
        if(root->right != nullptr) traversal(root->right, sum);
        root->val = root->val + sum;
        sum = root->val; //sum是直接就是当前节点的值了 因为当前节点的值 就是前面的累加值
        if(root->left != nullptr) traversal(root->left, sum);
    }
    TreeNode* convertBST(TreeNode* root) {
        //构造一个累加树 顺序是右中左累加就行了
        // 那么进行一个中序遍历 修改一下顺序就可以了
        int sum{0};
        traversal(root,sum);
        return root;
    }
};
```
