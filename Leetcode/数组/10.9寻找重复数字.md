# 寻找重复数字
使用一个抽象数组为链表的方法 这样子就转化为环形链表找入环口的问题了

环形链表需要两次相遇 
第一次是 快慢指针相遇找到相遇点
第二次是 起始点和相遇点相遇找到入环口

```cpp
class Solution {
public:
    int findDuplicate(vector<int>& nums) {
        // 数组抽象为一种链表 其中链表的下一个节点是i->nump[i]
        // 数组的索引i看作链表的「当前节点」。
        // 索引i对应的值（nums[i]）看作链表的「下一个节点的地址」。
        // 那么链表移动到下一个节点的逻辑是 i = nums[i]
        int slow{0};
        int fast{0};
        while(1) { 
            slow = nums[slow];
            fast = nums[nums[fast]];
            if(slow == fast) { // 相遇点和起始点同时移动 然后再相遇就是入环口
                slow  = 0;
                while(slow != fast) {
                    slow = nums[slow];
                    fast = nums[fast];
                }
                return slow;
            }
        }
        return 0;
    }
};
```
