# 找到出现一次的数字

第一个想法是使用哈希map去获取频率

但是在限制空间复杂度的时候不行了 那么使用一个异或运算去做
```cpp
class Solution {
public:
    int singleNumber(vector<int>& nums) {
        // 使用异或 我只能说牛逼
        int ans = 0;
        for(int i{0};i<nums.size();i++) {
            ans = ans ^ nums[i];
            // 真的牛逼 异或满足交换律和分配律
            // 因此没有必要纠结每一步的结果是什么 只需要知道最后结果一定会消除相同的两个的
        }
        return ans;
    }
};
```
